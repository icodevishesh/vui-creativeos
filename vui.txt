BUSINESS REQUIREMENT DOCUMENT (BRD)

Product Name (Working Title)
CreativeFlow / VUI CreativeOS

Objective
To build a Creative Agency Management Platform that streamlines:
Client onboarding 
Creative & content workflows 
Internal + external approvals 
Publishing automation 
Analytics & reporting 
Resource productivity tracking 

USER ROLES
Internal Users
1.Super Admin (Founder) 
2.Salesperson 
3.Account Manager 
4.Creative Lead / Team Lead 
5.Copywriter / Content Writer (Short + Long format: Social, Blogs, Website, SEO) 
6.Graphic Designer 
7.Video Editor 
External Users
1.Client Admin 
2.Client Reviewer 

MODULES OVERVIEW

1. CLIENT ONBOARDING MODULE
Inputs by Salesperson:
Basic Details:
Client Name 
Company Name 
Industry Category 
Client Admin Details (Name, Email, Phone) 

Scope of Work:
Engagement Type:
Retainer (Ongoing) 
Project-Based (Fixed Fee) 

Services Selection:
Social Media (Yes/No)
If Yes:
Platforms: 
oInstagram / Facebook / LinkedIn / YouTube / Twitter / Pinterest 
Deliverables Count (per platform) 
Content Split: 
oStatic / Video / Carousel (Optional) 

Paid Media (Yes/No)
If Yes:
Platforms: 
oMeta Ads 
oGoogle Ads 
oLinkedIn Ads 
oAmazon Ads 
oOthers 

Influencer Marketing (Yes/No)
Deliverable Count 
Platform selection 

Email Marketing (Yes/No)

Output:
Client Account Created 
Notification → Account Manager / Team Lead 

2. TEAM ALLOCATION MODULE
Assign: 
oAccount Manager 
oCreative Lead 
oCopywriter / Content Writer 
oDesigners 
oVideo Editors 
Access Control:
Only assigned team can view account data 

3. CLIENT ONBOARDING WORKFLOW
Actions:
Schedule onboarding call (email/calendar integration) 
Upload & Documentation:
Brand Guidelines 
Logos 
Fonts 
Brand Assets 
Structured Inputs:
Brand Positioning 
Target Audience 
Tone of Communication 
Month 1 Goals 
Onboarding Call Notes 
Requirement Gathering Notes / Meeting Logs 
Top 3 Competitors (for reference) 

4. PROJECT & TIMELINE MANAGEMENT
Detailed Gantt Chart:
Clear phase-wise breakdown: 
1.Onboarding 
2.Moodboard Creation 
3.Content Calendar Creation 
4.Internal Review 
5.Client Approval 
6.Design & Production 
7.Final Approval 
8.Publishing 
Key Features:
Visual clarity for client 
Milestone-based tracking 
Dependencies between tasks 
Deadlines & ownership clearly visible 

5. CONTENT CALENDAR MODULE
Workflow:
1.Account Manager creates timeline / Gantt chart 
2.Task auto-assigned → Copywriter / Content Writer (visible as OPEN TASK) 
3.Copywriter creates monthly content calendar 

Submission Flow:
Step 1: Internal Review
Submitted by Copywriter 
Reviewed by: 
oAccount Manager 
oCreative Lead 
Actions:
Approve 
Feedback 
Reject 

Step 2: Client Review (Only after internal approval)
Submitted by Account Manager 
Client Actions:
Approve 
Feedback 
Reject 

Status Flow:
Draft → Internal Review → Internal Approved → Client Review → Approved

6. CREATIVE PRODUCTION MODULE
Designers / Video Editors:
Receive approved content calendar automatically 
Tasks mapped with: 
oCopy 
oPlatform 
oPublishing date 

Workflow:
1.Upload creatives 
2.Submit → Internal Review 
3.Internal Approval: 
oAccount Manager 
oCreative Lead 
4.Submit → Client Review 
5.Client Approval → Auto Publishing 

7. APPROVAL SYSTEM (CORE ENGINE)
Approval Layers:
Internal 
Client 
Actions:
Approve 
Reject 
Feedback (comments) 
Tracking:
Iteration count 
Approval timelines 
Feedback logs 

8. AUTO PUBLISHING ENGINE
Integrations: 
oInstagram 
oFacebook 
oLinkedIn 
oYouTube 
Scheduled publishing 
Failure alerts/logs 

9. DASHBOARDS & ANALYTICS
Agency Dashboard:
Resource productivity: 
oAvg turnaround time 
oApproval success rate 
oIterations per task 
Bottleneck analysis 

Client Dashboard:
Deliverables vs Plan 
Approval timelines 
Engagement metrics: 
oReach 
oLikes 
oEngagement rate 

10. CREATIVE REPOSITORY
Storage: 3 months (auto-delete creatives) 
Logs retained permanently: 
oMetadata 
oApproval history 
oPerformance 

11. NOTIFICATION SYSTEM
Email + In-app alerts: 
oTask assignment 
oReview requests 
oFeedback updates 
oDeadlines 

12. ACTIVITY LOGGING
Timestamp tracking: 
oTask creation 
oSubmission 
oApproval 
oPublishing 

PERMISSIONS
Role-based access 
Account-level restriction 

FUTURE SCOPE
AI copy + content generation 
AI performance prediction 
Automated content suggestions 
SaaS monetization
